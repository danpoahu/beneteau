# Beneteau 373 Listing (Static)

This is a simple, static GitHub Pages site (no Firebase, no admin).

## Deploy
1. Put these files in your repo folder `/beneteau/` (or repo root if the repo is dedicated).
2. Ensure GitHub Pages is enabled for the repo.
3. Visit: https://danpoahu.github.io/beneteau

## Update content
- Photos are in `assets/`
- Bullet points + price are in `index.html`
